No terminal, após instalar o Nodejs.
Redirecione a saída do prompt para a pasta onde está o Nodejs:
(cd ~/pasta/nodejs-env)

Para ativar, digite:
source ./bin/activate

Depois de ativar, entre na pasta src
(cd nodejs-env/src)

Ative o arquico com:
(node index.js)

No navegador, digite:
(localhost:8007)
